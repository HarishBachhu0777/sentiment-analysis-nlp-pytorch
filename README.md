# Sentiment Analysis using PyTorch (LSTM)

## 📌 Overview
This project implements a sentiment classification model using PyTorch and LSTM networks on the SST2 dataset.

## 🚀 Features
- NLP preprocessing (tokenization, padding)
- Custom dataset pipeline
- LSTM-based deep learning model
- Binary sentiment classification

## 🧠 Model
- Embedding Layer
- LSTM Layer
- Fully Connected Layer

## 📊 Dataset
- Stanford Sentiment Treebank (SST2)
